# kikuchi-estate-csv-tool

キクチエステート通帳CSV取込→摘要付与→出力アプリ

通帳取引のCSVを取り込み、各取引に「摘要」を画面で入力し、CSVとして出力するブラウザツールです。インストール不要で、データはブラウザ内だけで処理されます（サーバーに送信されません）。

## 使い方

1. `index.html` をダブルクリックしてブラウザで開く
2. 通帳データを取り込む（2つの方法から選べます）
   - **CSVファイルから**: 通帳CSVをドラッグ＆ドロップ（または「ファイルを選択」）。Shift-JIS / UTF-8 を自動判別します
   - **コピー＆貼り付けから**: ネットバンキングの取引明細をマウスで選択してコピーし、貼り付け欄に貼って「取り込む」（タブ区切り／カンマ区切りを自動判別）
   - 見出し行が無いデータは「1行目を見出しとして扱う」のチェックを外してください
3. 黄色の「摘要」列のセルをクリックして摘要を入力する（不要な行は ✕ で削除）
4. 出力ファイル名を指定して「CSVをダウンロード」を押す
   - 元データの全列＋入力した摘要をまとめて出力します
   - Excelで開いても文字化けしないUTF-8(BOM付き)で出力されます

## 対応する通帳データの形式

列の構成は問いません（表示した全列をそのまま保持し、摘要列だけ追加・編集します）。以下のようなパターンに対応しています。

- 日付・出金・入金・残高・摘要 のように入金/出金が別列のもの
- 入出金が1列で、プラス/マイナスで入金・出金を表すもの

## 構成

- `index.html` … 画面
- `styles.css` … 見た目
- `app.js` … CSV読込・摘要編集・CSV出力の処理

## 補足

- 取り込んだCSVに既に「摘要」を含む列があればその列を編集対象にし、無ければ新しく「摘要」列を追加します。
- 実際の通帳CSVの形式に合わせて、列の扱いなどは今後調整できます。
