# kikuchi-estate-csv-tool

キクチエステート通帳CSV取込→摘要付与→出力アプリ

通帳取引のCSV（またはネットバンキング画面のコピー）を口座ごとに取り込んで蓄積し、各取引に「法人/個人の区分」と「入出金の内訳（家賃・ガス代・預り敷金など）」を登録して、口座ごとのCSVとして出力するブラウザツールです。インストール不要で、データはお使いのブラウザ内だけに保存されます（サーバーには送信されません）。

## 使い方

1. `index.html` をダブルクリックしてブラウザで開く
2. **口座を選ぶ**: 画面上部のタブで口座を切り替え。初回は「＋ 口座を追加」で銀行名・口座番号を登録します
3. **通帳データを取り込む**（2つの方法から選べます）
   - **CSVファイルから**: 通帳CSVをドラッグ＆ドロップ（または「ファイルを選択」）。Shift-JIS / UTF-8 を自動判別します
   - **コピー＆貼り付けから**: ネットバンキングの取引明細をマウスで選択してコピーし、貼り付け欄に貼って「取り込む」（タブ区切り／カンマ区切りを自動判別）
   - 見出し行が無いデータは「1行目を見出しとして扱う」のチェックを外してください
4. **列の対応を設定**（初回のみ）: どの列が「日付/出金/入金/入出金/残高/摘要」かを選びます。設定内容は口座ごとに記憶され、次回からは自動で適用されます（「列の対応を変更する」でいつでも変更可）
5. **明細の確認・分類の入力**: 各行の「分類を編集」から、法人/個人の区分と、入出金の内訳（分類＋金額）を登録します。入力は自動保存されます
6. **CSVを出力**: 画面右上の「⚙ 設定」→「全口座データの一括ダウンロード」。期間（年月〜年月）を指定し、口座ごとのCSVをダウンロードします（内訳は1分類につき1行で出力）

## 主な機能

- **口座の登録・選択**: 銀行名と口座番号を登録し、Chromeタブ風のタブで切り替え。口座ごとに明細を蓄積
- **入出金の分類**: 各取引に「法人/個人」と内訳を登録
  - 入金: 家賃 / ガス代 / 預り敷金 / その他
  - 出金: 預り敷金の返済 / 家賃の返金 / ガス代の返金 / その他
  - 1つの取引を複数分類に分けて金額を内訳登録できます
- **分類の学習**: 同じ「摘要×区分（入金/出金）」が再び現れたら、前回の内訳をあらかじめ表示します。取引額と内訳合計が一致しない場合は赤字で警告します
- **自動保存**: 取り込んだ明細と入力した分類はブラウザ内に自動保存
- **重複の自動除外**: 同じ日のデータを誤って二重に取り込んでも、「日付・入出金・残高・摘要（取引内容）」が一致する行は重複登録されません
- **列マッピングの学習**: 取込時に設定した列の対応を口座ごとに記憶
- **一括ダウンロード（期間指定）**: 全口座を口座ごとのCSVで出力。年月〜年月の期間指定が可能
- **バックアップ／復元**: 全データをファイルに書き出し（バックアップ）／読み戻し（復元）

## 対応する通帳データの形式

列の構成は問いません（全列をそのまま保持し、摘要列だけ追加・編集します）。以下のようなパターンに対応しています。

- 日付・出金・入金・残高・摘要 のように入金/出金が別列のもの
- 入出金が1列で、プラス/マイナスで入金・出金を表すもの

## データの保存場所と注意

- データは「お使いのブラウザの中」に保存されます。**別のパソコンや別のブラウザとは共有されません**。また、ブラウザのデータ（履歴・サイトデータ）を消すと保存内容も消えます。
- パソコンの買い替えやデータ削除に備えて、ときどき「バックアップを保存」でファイルに書き出しておいてください。復元は「バックアップから復元」で行えます。

## 補足

- 一括ダウンロードで複数の口座がある場合、ブラウザが「複数ファイルのダウンロードを許可しますか？」と確認することがあります。「許可」を選んでください。
- 期間でのしぼり込みは、日付欄に4桁の西暦（例: 2026/05/01）が含まれていることを想定しています。和暦のみの場合は期間指定で除外されず全件出力されます。
